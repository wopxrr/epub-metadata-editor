Saya memiliki file EPUB yang metadata-nya kurang lengkap. Buku tersebut berjudul: **[JUDUL BUKU]**.

Tolong cari informasi metadata lengkap untuk buku ini dari sumber seperti Google Books, Goodreads, Amazon, Library of Congress, atau Open Library. Kembalikan hasilnya dalam format JSON yang rapi dan presisi seperti di bawah ini.

## Format Output yang Diinginkan (JSON)

```json
{
  "title": "Judul Lengkap Buku",
  "title_sort": "Judul untuk sorting (hilangkan The/A/An di depan)",
  "creators": ["Nama Penulis Utama", "Nama Penulis Kedua (jika ada)"],
  "author_sort": "Belakang, Depan (contoh: Lewis, C. S.)",
  "series": "Nama Seri/Series (kosong jika standalone)",
  "series_index": "Nomor urut dalam seri (contoh: 1, 2, 3)",
  "description": "Sinopsis lengkap dalam HTML (gunakan tag <p>, <em>, <h3> jika perlu)",
  "language": "Kode BCP-47 (contoh: en, id, fr, de, ja)",
  "publisher": "Nama Penerbit",
  "date": "Tanggal Publikasi (ISO 8601: YYYY-MM-DD atau YYYY-MM-DDTHH:MM:SS+00:00)",
  "subjects": ["Genre 1", "Genre 2", "Genre 3"],
  "identifiers": {
    "isbn": "ISBN-10 atau ISBN-13 (tanpa tanda hubung)",
    "isbn_13": "ISBN-13 lengkap",
    "asin": "ASIN Amazon (jika ada)",
    "uuid": "UUID unik (generate UUID v4 baru jika tidak ditemukan)"
  },
  "rating": "Rating rata-rata dari Goodreads/Amazon (skala 1-10, contoh: 8.0)",
  "modification_date": "Tanggal terakhir metadata ini di-update (gunakan tanggal hari ini)"
}
```

## Catatan Penting:

1. **title_sort**: Hilangkan kata depan seperti "The", "A", "An" dan pindahkan ke belakang dengan koma.
   - Contoh: `"The Hobbit"` → `"Hobbit, The"`
   - Contoh: `"A Game of Thrones"` → `"Game of Thrones, A"`

2. **author_sort**: Format harus `"Belakang, Depan"`.
   - Contoh: `"J. K. Rowling"` → `"Rowling, J. K."`
   - Contoh: `"C. S. Lewis"` → `"Lewis, C. S."`

3. **description**: Buat sinopsis yang informatif. Gunakan HTML tags yang sesuai:
   - `<p>` untuk paragraf
   - `<em>` untuk judul buku di dalam teks
   - `<h3>` untuk sub-judul seperti "Review" atau "About the Author"
   - Hindari bullet points; gunakan paragraf saja.

4. **identifiers**: Jika ISBN tidak ditemukan, buatkan UUID v4 yang valid. Jangan biarkan field ini kosong.

5. **subjects**: Gunakan 3–5 tag genre yang relevan. Contoh: `["Juvenile Fiction", "Fantasy & Magic", "Classics"]`.

6. **language**: Gunakan kode BCP-47 standar:
   - `en` = English
   - `id` = Indonesian
   - `fr` = French
   - `de` = German
   - `ja` = Japanese
   - `zh` = Chinese
   - `ko` = Korean

7. **date**: Tanggal PUBLIKASI buku, bukan tanggal file dibuat.

8. **rating**: Kalau tidak ada rating yang jelas, biarkan kosong string `""`.

9. **modification_date**: Gunakan tanggal hari ini dengan format ISO 8601.

## CARA PENGGUNAAN:

1. Salin seluruh teks di atas ke chat AI (ChatGPT, Claude, Gemini, dll).
2. Ganti `[JUDUL BUKU]` dengan judul buku yang ingin kamu cari.
3. Kirim prompt-nya.
4. Copy hasil JSON-nya.
5. Di EPUB Metadata Editor, paste ke field yang sesuai. Untuk field `identifiers`, copy setiap ID dan pisahkan dengan `;` di form Identifiers.

## Tips:
- Jika AI tidak bisa generate UUID, gunakan website: https://www.uuidgenerator.net/
- Jika rating tidak ada, kosongkan saja.
- Untuk field `subjects` di aplikasi, formatnya adalah tag dipisahkan dengan `;` (semicolon).
