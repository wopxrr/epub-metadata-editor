# PROMPT: Find Complete Book Metadata for EPUB
# Copy the text below and send it to AI (ChatGPT, Claude, Gemini, etc.)
# Replace "[BOOK TITLE]" with the title of the book you want to search

---

I have an EPUB file with incomplete metadata. The book title is: **[BOOK TITLE]**.

Please search for complete metadata for this book from sources such as Google Books, Goodreads, Amazon, Library of Congress, or Open Library. Return the result in a clean, precise JSON format as shown below.

## Desired Output Format (JSON)

```json
{
  "title": "Full Book Title",
  "title_sort": "Title for sorting (remove leading The/A/An)",
  "creators": ["Primary Author", "Secondary Author (if any)"],
  "author_sort": "Last, First (example: Lewis, C. S.)",
  "series": "Series Name (empty if standalone)",
  "series_index": "Number in series (example: 1, 2, 3)",
  "description": "Full synopsis in HTML (use <p>, <em>, <h3> if needed)",
  "language": "BCP-47 code (example: en, id, fr, de, ja)",
  "publisher": "Publisher Name",
  "date": "Publication Date (ISO 8601: YYYY-MM-DD or YYYY-MM-DDTHH:MM:SS+00:00)",
  "subjects": ["Genre 1", "Genre 2", "Genre 3"],
  "identifiers": {
    "isbn": "ISBN-10 or ISBN-13 (no hyphens)",
    "isbn_13": "Full ISBN-13",
    "asin": "Amazon ASIN (if available)",
    "uuid": "Unique UUID (generate new UUID v4 if not found)"
  },
  "rating": "Average rating from Goodreads/Amazon (scale 1-10, example: 8.0)",
  "modification_date": "Date this metadata was last updated (use today's date)"
}
```

## Important Notes:

1. **title_sort**: Remove leading words like "The", "A", "An" and move them to the end with a comma.
   - Example: `"The Hobbit"` → `"Hobbit, The"`
   - Example: `"A Game of Thrones"` → `"Game of Thrones, A"`

2. **author_sort**: Format must be `"Last, First"`.
   - Example: `"J. K. Rowling"` → `"Rowling, J. K."`
   - Example: `"C. S. Lewis"` → `"Lewis, C. S."`

3. **description**: Write an informative synopsis. Use appropriate HTML tags:
   - `<p>` for paragraphs
   - `<em>` for book titles within the text
   - `<h3>` for sub-headings like "Review" or "About the Author"
   - Avoid bullet points; use paragraphs only.

4. **identifiers**: If no ISBN is found, generate a valid UUID v4. Do not leave this field empty.

5. **subjects**: Use 3–5 relevant genre tags. Example: `["Juvenile Fiction", "Fantasy & Magic", "Classics"]`.

6. **language**: Use standard BCP-47 codes:
   - `en` = English
   - `id` = Indonesian
   - `fr` = French
   - `de` = German
   - `ja` = Japanese
   - `zh` = Chinese
   - `ko` = Korean

7. **date**: The BOOK PUBLICATION date, not the file creation date.

8. **rating**: If no clear rating exists, leave as empty string `""`.

9. **modification_date**: Use today's date in ISO 8601 format.

## HOW TO USE:

1. Copy the entire text above into the AI chat (ChatGPT, Claude, Gemini, etc.).
2. Replace `[BOOK TITLE]` with the title of the book you want to search.
3. Send the prompt.
4. Copy the resulting JSON.
5. In EPUB Metadata Editor, paste into the appropriate fields. For the `identifiers` field, copy each ID and separate them with `;` in the Identifiers form.

## Tips:
- If the AI cannot generate a UUID, use: https://www.uuidgenerator.net/
- If there is no rating, leave it empty.
- For the `subjects` field in the app, the format is tags separated by `;` (semicolon).
